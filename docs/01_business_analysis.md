# 01 — Business Analysis

## 1. Industry Context

The platform serves **printing, advertising, branding, publishing, marketing,
educational printing, outdoor advertising and embroidery** companies. These
businesses share a common operational pattern:

1. **Client-driven custom production** — every order is unique (size, material,
   design, deadline).
2. **Heavy file workflows** — designs travel from customer ↔ designer ↔ press
   in many revisions.
3. **Approval loops** — legally meaningful "sign-off" before production starts.
4. **Multi-discipline production** — same order may touch design, large-format
   print, finishing, embroidery, and on-site installation.
5. **Inventory of raw materials** — flex, vinyl, paper, ink, thread, acrylic,
   PVC, etc. — must be tracked per job.
6. **Field operations** — billboard installation, vehicle wrapping, signboard
   installation are off-premises projects.

## 2. Stakeholders

| Stakeholder         | Primary Concern                                   |
| ------------------- | ------------------------------------------------- |
| CEO                 | Live company-wide visibility, profitability       |

| Print Operator      | Print queue, materials, machine assignments       |

| Sales Employee      | Leads, opportunities, follow-ups, quotations      |

| Customer            | Self-service ordering, file upload, approvals     |

## 3. Pain Points Solved

- Lost designs / unclear revisions → **versioned file management + approvals**.
- Verbal / WhatsApp orders → **structured online ordering with instant pricing**.
- No production visibility → **live production tracking & monitoring**.
- Manual quotations → **automatic price calculator + signable quotations**.
- Inventory leakage → **stock-in / stock-out / damaged tracking**.
- "Who did what?" disputes → **full audit log + employee timelines**.
- Owner blind-spot → **Executive Command Center**.

## 4. Business Value

- 30–50% reduction in order intake time (online ordering + instant quote).
- Faster cash flow (digital quotations + signature + linked invoice).
- 100% audit traceability — legally defensible approval history.
- Higher customer satisfaction (portal, tracking, transparent communication).
- Data-driven management (analytics across revenue, productivity, inventory).

## 5. Strategic Goals

1. **Single source of truth** across sales, production, inventory, finance.
2. **Customer self-service** to scale without scaling headcount linearly.
3. **Executive monitoring** so leadership decisions are data-backed.
4. **Compliance-ready** record-keeping (approvals, signatures, audit log).
5. **Modular architecture** so vertical add-ons can ship without regressions.
