# 04 — User Stories (sample)

Format: **As a [role] I want [action] so that [value].**

## CEO

- As a CEO I want a live dashboard of revenue, orders and people online
  so that I can manage the company in real time.
- As a CEO I want to drill from an audit log entry to the affected record
  so that I can investigate incidents instantly.

## General Manager

- As a GM I want to see every department's workload and SLA so that I can
  rebalance resources before deadlines slip.

## Sales Employee

- As a Sales rep I want to convert a Lead into a Customer + Opportunity with
  one click so that I never lose a deal during data entry.
- As a Sales rep I want to send a digital quotation to a customer and be
  notified when they accept it so that I can hand off to production immediately.

## Designer

- As a Designer I want to upload a new revision and request the customer's
  approval so that I have a legally clear sign-off before printing.

## Print Operator

- As an Operator I want a print queue ordered by deadline + priority so that
  I always pick the next correct job.
- As an Operator I want to log material usage on a job so that inventory is
  accurate without manual end-of-day reports.

## Accountant

- As an Accountant I want to generate an invoice directly from an accepted
  quotation so that nothing is re-typed.
- As an Accountant I want to record a partial payment and see the customer
  balance recompute automatically.

## Warehouse Employee

- As a Warehouse employee I want to log a damaged-material event with a
  photo so that shrinkage is auditable.

## Marketing Employee

- As a Marketing employee I want a content calendar across all social
  accounts so that nothing is double-posted.

## Customer Support

- As a Support agent I want all customer messages from email, portal and
  tickets in one inbox so that no message is missed.

## Customer

- As a Customer I want to place a new printing order online with an instant
  price estimate so that I get clarity before committing.
- As a Customer I want to upload my source files and reference designs while
  placing the order so that the design team has everything immediately.
- As a Customer I want to approve a design and sign the approval with my
  finger / mouse so that I do not need to print and scan a PDF.
- As a Customer I want to see all my invoices and outstanding balance in one
  place.

## Designer Monitoring (Manager view)

- As a Department Manager I want to see today's productivity per designer
  (designs created, revisions, approvals, average completion time) so that
  I can reward and coach fairly.
